## 실행 방법

### 1. git clone 을 통해 repository를 local directory 로 복사

### 2. client 폴더와 server 폴더에서 각각 npm install 을 통해 module 설치

### 3. Backend의 DB 연동을 위한 `.env` 파일 설정 추가
```
MONGO_URI=< 몽고DB URI >
PORT=5001
```
### 4. client 와 server 폴더를 각각 npm run start 로 실행
